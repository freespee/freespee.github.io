FreespeeDelegate

@protocol FreespeeDelegate <NSObject>

Delegate for the Freespee SDK.

  • -freespee:handleIncomingCall:

    Called when there is an incoming call to the app. All interaction and monitoring of the call is done via the supplied call instance.

    Note

    Set yourself as a listener on the call to observe future state changes. Also consider the application state as the call comes in.

    Declaration

    Objective-C

    - (void)freespee:(nonnull Freespee *)freespee
    handleIncomingCall:(nonnull id<FreespeeCall>)call;

    Parameters

    freespee

    Freespee instance.
    call

    Call handle.
  • -freespee:provideUserCredential:

    Called as part of registration - requesting a credential for the user being connected.

    Declaration

    Objective-C

    - (void)freespee:(nonnull Freespee *)freespee
    provideUserCredential:(nonnull void (^)(NSString *_Nullable))callback;

    Swift

    func freespee(_ freespee: Freespee, provideUserCredential callback: @escaping (String?) -> Void)

    Parameters

    callback

    Must be called with a credential or nil upon failure to provide a credential.
