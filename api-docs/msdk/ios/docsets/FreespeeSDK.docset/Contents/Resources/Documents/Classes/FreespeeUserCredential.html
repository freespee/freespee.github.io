FreespeeUserCredential

@interface FreespeeUserCredential : NSObject

Convenience class for generating a Freespee user credential in-app. It is not recommended to use this approach in production as the secret can be leaked. Instead generate the credential in your app backend.

  • +generateWithSecret:userIdentifier:error:
