  • -startListeningForIncomingCalls:

    Sets the SDK in a state where it listens for incoming phone calls without the need for recieving a VoIP push notification.

    Note

    You should use this for debugging/testing purposes only, using this in production will consume significant battery life.

  • -handleIncomingNotification:

    Handle an incoming PushKit VoIP push notification. To be called when receiving a push to the PKPushRegistry, before the completion block is executed.

    Note

    On valid push, -freespee:handleIncomingCall: on FreespeeDelegate will be called synchronously. Signal a new incoming call to CallKit within this callback. If you fail to do so on iOS 13, your app will be terminated.

    Declaration
